COMPONENTES SWING UTILIZADOS NA ReceitaView.java
================================================

Estrutura base (containers)
---------------------------
1. `JFrame ReceitaView` – janela principal.
2. `JScrollPane ScrollPane` – envolve todo o painel para permitir rolagem vertical.
3. `JPanel MainPanel` – painel principal com `GroupLayout` que acomoda os campos.

Cabeçalho / Identificação
-------------------------
4. `JLabel TituloLabel` – título “Criar Nova Receita”.

Entradas de texto
-----------------
5. `JLabel NomeLabel` + `JTextField NomeTextField` – nome da receita.
6. `JLabel DescricaoLabel` + `JTextArea DescricaoTextArea` dentro de `JScrollPane DescricaoScrollPane` – descrição.
7. `JLabel ModoPreparoLabel` + `JTextArea ModoPreparoTextArea` em `JScrollPane ModoPreparoScrollPane` – modo de preparo, com `lineWrap`.

Seleção de categoria
--------------------
8. `JLabel CategoriaLabel`.
9. `JComboBox<String> CategoriaComboBox` – itens preenchidos via `configureCategoryComboBox()`.

Bloco de ingredientes
---------------------
10. `JLabel IngredientesLabel`.
11. `JComboBox<String> IngredienteComboBox` – lista com nome + unidade (carregada de `IngredienteController`).
12. `JLabel QuantidadeLabel`.
13. `JTextField QuantidadeTextField`.
14. `JButton AdicionarIngredienteButton` – chama `addIngredientToRecipe`.
15. `JButton RemoverIngredienteButton` – chama `removeIngredientFromRecipe`.
16. `JTable IngredientesTable` dentro de `JScrollPane IngredientesScrollPane` – usa `DefaultTableModel` com colunas “Ingrediente”, “Quantidade”, “Unidade”.

Ações finais
------------
17. `JButton CriarButton` – dispara `createRecipeAndPersist`.

Comportamento adicional
-----------------------
- Look and feel definido para “Nimbus” no `main`.
- Construtor recebe `UsuarioModel` para saber quem está logado.
- `WindowListener` fecha a tela voltando ao `MenuView`.

Utilize exatamente esses componentes/tipos ao remontar a tela no NetBeans GUI Builder para manter compatibilidade com o código atual.
